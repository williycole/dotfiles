# Dotfiles

Personal configuration files for multiple platforms.

## Structure

```
dotfiles/
├── omarchy/      # Arch Linux with Hyprland
├── ubuntu/       # Ubuntu/Debian
├── windows/      # Windows
├── wsl/          # Windows Subsystem for Linux
└── backgrounds/  # Wallpapers
```

## Configurations

| Platform | Included |
|----------|----------|
| **omarchy** | hypr, ghostty, nvim, lazygit, zsh/bash, scripts |
| **ubuntu** | nvim, ghostty, lazygit, zsh/bash, dlv, scripts |
| **windows** | nvim, glazewm, zebar, alacritty, wezterm, oh-my-posh, PowerShell |
| **wsl** | nvim, lazygit, zsh/bash, scripts |

## Usage

Copy the relevant configs to your home directory or symlink them as needed.
